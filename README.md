# Popcorn-BE
Popcorn is a website where you can search for types of media and filter them by genre, and add them to a watchlist with different states.

***
### ***Technologies Used*** 
- React
- MongoDB
- Express
- Node.js
- Vite
- JWT Auth
- CSS
- Insomnia
***
### ***Getting started***
- A visitor can view and see the Genre list page, and the Genre pages.
- A vistor should sign-up to the website in order to get access for features.
- When the user signs in, it can create a watchlist to and add different types of media.
